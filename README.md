Node version >= 8.0

# Node API Skeleton #

## Installation

Install dependencies

```
npm install
```

## Running

```
npm start
```

or watch it with nodemon

```
npm run watch
```

or, for a more verbose output:

```
npm run verbose
```

as a convenience to --inspect:

```
npm run inspect
```

## Testing

Run all tests:

```
npm test
```

Run unit tests only:

```
npm run test:unit
```

Run functional tests only:

```
npm run test:functional
```
